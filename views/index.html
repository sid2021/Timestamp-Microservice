Timestamp API

This project is based on Freecodecamp's API and Microservices certification.. It is an API which allows to get Unix timestamps and UTC dates in a JSON format.

Example usage

Get the Current Date

To get the data for the current date go to the following route:

/api/timestamp

Get the Chosen Date

To obtain the data for a chosen date add the date (YYYY-MM-DD format) or a unix timestamp to the route: 

/api/timestamp/2020-06-15

Unix timestamp has to consist of numbers only:

/api/timestamp/123987456

Example Response

A successful response should return a unix timestamp and its associated UTC date in a JSON format: 


        {
          "unix":1601078400000,
          "utc":"Sat, 26 Sep 2020 00:00:00 GMT"
        }
